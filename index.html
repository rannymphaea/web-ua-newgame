Pillar Game Logic

Dasar pembuatan game yang pertama adalah program! Di Pillar Game Logic kamu belajar lebih lanjut tentang pemrograman, fisika dalam game, dan bagaimana kode bisa mempengaruhi game yang ingin kamu buat.

Pillar Game Design

Pembuatan game bukan hanya soal kode — design juga sangat menentukan. Di Pillar Game Design kamu belajar cara mendesain UI dalam game, serta bagaimana karakter bergerak secara 2D maupun 3D.

Pillar Game Sound

Pernah main game tanpa suara? Terdengar hampa kan! Di sini kamu belajar sound effect, music background, dan cara mengisi suara karakter agar game benar-benar terasa hidup.

Quest

Quest merupakan divisi-divisi yang ada di NewGame. Di masing-masing quest terdapat task dan jobdesk tersendiri, serta Core Quest (Master Quest, Commander, Quest Keeper, Gold Guardian) untuk mengaturnya.

Geser (Swipe) atau Tekan kartu di bawah untuk melihat divisi lainnya:

⚔ FamilyGame Quest
Biro kekeluargaan yang mengatur healing anggota dan mengawasi quest-quest lainnya.
🎨 Designcraft Quest
Divisi medinkraf — mengatur postingan social media NewGame, membuat sertifikat dan design yang dibutuhkan.
🤝 Alliance Quest
Divisi yang bertugas mencari, membuat, dan menjaga hubungan dengan pihak sponsor dan partner.
📦 Inventory Quest
Divisi inventaris — mengatur dan menjaga barang serta aset NewGame, serta kebersihan ruangan yang dipakai.
🌟 Newmember Quest
Divisi open recruitment yang bertugas mengatur dan menyelenggarakan pencarian anggota baru setiap tahunnya.
⚡ Training Quest
Divisi pelatihan skill — menyelenggarakan pelatihan membuat game dan mengawasi perkembangan skill seluruh anggota sesuai pillarnya.
🚀 Project Quest
Divisi program kerja — mengatur dan mengelola proyek NewGame ke depannya, seperti membuat game ataupun menyelenggarakan GameJam.

Sistem Penilaian di NewGame

Sistem kerja di NewGame itu berdasarkan game lhoo! Anggota mengerjakan task di masing-masing quest dan mendapatkan poin EXP.

Skor EXP sangat penting karena menentukan jabatan Core & Quest, Penghargaan Anggota Terbaik tiap bulan, dan yang paling seru — siapa yang berangkat lomba GameJam ke Pulau Jawa atau Luar Negeri!

Pembagian Task (diberikan berdasarkan task yang dikerjakan):

1. Task Keorganisasian — aktivitas dari quest masing-masing.
2. Task yang berkaitan dengan game — kontribusi nyata terhadap pengembangan game (coding, desain, ide, testing, dll).
3. Task Request — tugas yang ingin dibagikan ke quest yang lain.

Tingkatan Anggota Terdiri Dari:

🔥 Anggota Fighter — Anggota on fire! Aktif dan punya EXP tertinggi.
🎮 Anggota Biasa — EXP rata-rata dan kadang-kadang izin.
👻 Anggota NPC — Belum bisa aktif karena sibuk. Pergerakannya seperti NPC!

Jenis-Jenis Anggota:

🔰 Trainee
Anggota weekly training yang belum eligible/berhak menyentuh project apapun.
🤝 Associate
Boleh menyentuh project, tapi hanya tugas-tugas kecil yang tidak memberikan influence.
⚔️ Soldat
Pemain inti yang berhak masuk ke project GOTS/Group Project.

*PS: Seluruh rank masih boleh mengikuti study weekly.

Cara Mengetahui Anggota Eligible:

1. Rajin datang di training (absensi training akan dinilai).
2. Selalu buat tugas training (Trainer memberikan nilai pada tiap murid, nilai akan dikumpulkan untuk penilaian akhir).
3. Untuk naik Rank, Trainee harus mengumpulkan nilai (nilainya akan dihitung).

Project Idea

Di NewGame, kita punya berbagai tingkatan project untuk melatih skill dan mewujudkan ide-ide liar kamu menjadi game yang nyata!

🕹️

Alpha Project

Project game kecil-kecilan (sekelas Flappy Bird atau game simpel lainnya). Project ini beranggotakan para Associate yang baru menyentuh project secara langsung. Project ini dilakukan per kelompok dan hanya untuk belajar melakukan project.
👥

Beta Project

(Group Project)
Para Soldat awal boleh membuat/bergabung ke grup mengenai project yang ingin dibuat (didiskusikan saat rapat). Game yang dibuat akan di-publish ke media kecil-kecilan (itch.io) atau media besar jika diinginkan.
👑

GOTS

Ini project Utama dan fokus inti dari NewGame. Game yang dibuat di sini harus selesai dan boleh sekompleks apapun. Anggota dari GOTS ini boleh dari siapapun yang sudah mencapai rank Soldat.
